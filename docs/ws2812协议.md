# WS2812 单线归零码协议

## 概述

WS2812 是 Worldsemi 开发的智能控制 LED 芯片，集成了控制电路和 RGB 发光芯片。每个 LED 内部有 24 位移位寄存器和输出锁存器，通过 **单线归零码**（NRZ）协议级联通信。

## 物理层

- 一根数据线 DIN：单向数据输入
- DOUT 级联到下一个 WS2812 的 DIN
- 使用 **RMT（Remote Control）** 外设驱动，以 0.1μs（10MHz）精度生成精确波形
- 空闲状态：低电平
- 供电：VDD=3.5~5.3V，典型 5V

## 通信时序

所有时序由 RMT 硬件精确控制，**无需 CPU 干预**。

### 码元定义（10MHz 分辨率，每 tick = 0.1μs）

| 码元 | 高电平 | 低电平 | 波形 |
|------|--------|--------|------|
| **0 码** | 0.3μs (3 ticks) | 0.9μs (9 ticks) | `┌──┴──────┘` |
| **1 码** | 0.9μs (9 ticks) | 0.3μs (3 ticks) | `┌──────┴──┘` |

```
0 码:  ┌────┐                              ┌────
       │    │  0.9μs                       │
       └────┘  0.3μs                      ↑

1 码:  ┌──────────┐                        ┌────
       │          0.3μs                   │
       └──────────┘ 0.9μs                ↑
```

### 复位码

```
      ┌──────────────────────────────────────┐
      │  拉低 ≥50μs（实际 60μs）             │
      └──────────────────────────────────────┘
```
传输完 24 位数据后，拉低 50μs 以上触发锁存，LED 显示新颜色。

## 数据传输

- **高位优先**：每个字节先传 MSB（bit7），后传 LSB（bit0）
- 24 位数据排列：**G-R-B**（Green 最高位，Blue 最低位）

```
第一个字节        第二个字节        第三个字节
Bit7─────Bit0   Bit7─────Bit0   Bit7─────Bit0
 G7 G6 G5 G4 G3 G2 G1 G0   R7 R6 R5 R4 R3 R2 R1 R0   B7 B6 B5 B4 B3 B2 B1 B0
```

## 级联方式

```
主机 →──── DIN     DOUT ────→ DIN     DOUT ────→ ...
          LED #1             LED #2
```

- 主机依次发送所有 LED 的数据帧
- 第一帧被 LED#1 锁存，后续数据移入内部移位寄存器
- 每个 LED 收到 24 位后，后续数据自动转发到 DOUT
- 全部发送完成后，发送 ≥50μs 复位码，所有 LED 同时更新

## 颜色数据格式

### RGB 888（24 位）

每个通道 8 位，共 256 级灰度：

| 通道 | 范围 | 说明 |
|------|------|------|
| Green | 0~255 | 内部存储顺序第一位 |
| Red | 0~255 | 内部存储顺序第二位 |
| Blue | 0~255 | 内部存储顺序第三位 |

### HSV 模型

RGB 颜色空间不易调节色相，代码内置 HSV→RGB 转换：

| 参数 | 范围 | 说明 |
|------|------|------|
| H（色相） | 0~359° | 0=红，120=绿，240=蓝 |
| S（饱和度） | 0~100 | 0=灰，100=纯色 |
| V（明度） | 0~100 | 0=黑，100=全亮（受亮度限制） |

## 亮度控制

驱动器通过软件线性缩放实现全局亮度：

```
实际 R = (R × brightness) / 100
实际 G = (G × brightness) / 100
实际 B = (B × brightness) / 100
```

| 等级 | brightness 值 |
|------|---------------|
| 关 | 0 |
| 默认 | 30 |
| 最大 | 100 |

## 软硬件架构

```
┌─────────┐    RMT 发送     ┌──────────────┐
│ 应用层   │ ──────────→   │  RMT 外设     │
│ rgb.c    │               │  10MHz 时钟   │
└─────────┘               └──────┬───────┘
                                 │
                          ┌──────▼───────┐
                          │ 编码管道      │
                          │              │
                          │ bytes_encoder│  0/1 码 → 24bits
                          │ copy_encoder │  复位码 ≥50μs
                          └──────┬───────┘
                                 │
                          ┌──────▼───────┐
                          │  TX 通道     │
                          │  GPIO16      │
                          └──────────────┘
```

## 驱动 API

| 函数 | 说明 |
|------|------|
| `rgb_init()` | 初始化 RMT TX 通道，安装编码器，默认熄灭 |
| `rgb_set_color(r, g, b)` | 设置 RGB 颜色（8 位每通道），自动应用亮度缩放 |
| `rgb_set_hsv(h, s, v)` | 设置 HSV 颜色，内部转换为 RGB |
| `rgb_set_brightness(percent)` | 设置亮度 0~100，默认 30 |
| `rgb_clear()` | 熄灭 LED（三通道归零） |
