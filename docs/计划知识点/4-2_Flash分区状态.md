## 分区表（`partitions.csv`）

| 分区名 | 类型 | 子类型 | 偏移 | 大小 | 用途 |
|--------|------|--------|------|------|------|
| nvs | data | nvs | `0x9000` | **20KB** | Wi-Fi 配置、校准数据（NVS key-value） |
| phy_init | data | phy | `0xE000` | **4KB** | PHY 初始化参数（射频校准数据） |
| factory | app | factory | `0x10000` | **2MB** | 应用程序固件（当前运行） |
| storage | data | spiffs | 自动 | **2MB** | 配置文件（WiFi 配置、校准参数等） |

### 偏移说明

```
0x0000 ┌─────────────────────────┐
       │   bootloader (3 sectors) │   ① 偏移 0x0000，固定占用 0x9000 字节
0x9000 ├─────────────────────────┤
       │   nvs (20KB)            │   ② NVS 分区，WiFi 等标准配置
0xE000 ├─────────────────────────┤
       │   phy_init (4KB)        │   ③ 射频参数
0x10000├─────────────────────────┤
       │   factory app (2MB)     │   ④ 固件主体，从 0x10000 开始
0x210000├─────────────────────────┤
       │   storage / SPIFFS (2MB) │  ⑤ 配置文件系统，Offset 自动分配
0x410000└─────────────────────────┘
         ↑ 总 Flash 16MB，剩余约 12MB 未使用
```

## 各文件体积构成

```
hello_world.bin (301 KB)
  ├─ .text（代码段）        ~180 KB  — 程序指令
  ├─ .rodata（只读数据）     ~60 KB  — 字符串、常量
  ├─ .data（已初始化数据）    ~5 KB   — 全局变量初始值
  ├─ .bss（未初始化数据）     0 KB    — 运行时清零（不占固件空间）
  └─ 分区头 / 校验           ~10 KB

elf 文件 4.5MB（含调试符号，bin 不含）
map 文件 2.9MB（链接映射表，仅用于调试）
```
